で話は戻る。



俺はxorg.confに<a href="http://d.hatena.ne.jp/flageo/20071125/p1" target="_blank">以下のように</a>設定していたわけだ。

Option          &#34;XkbRules&#34;      &#34;xorg&#34;
Option          &#34;XkbModel&#34;      &#34;jp106&#34;
Option          &#34;XkbLayout&#34;     &#34;jp&#34;
#CTRLとCAPSLOCK入れ替え
Option          &#34;XkbOptions&#34;    &#34;ctrl:swapcaps&#34;

これをXorg7.4でもXKBに伝えてやればいいのだが、方法は3つある。



1. haldを通して伝える。



2. haldを通さず、xorg.confから伝える。



3. haldもxorg.confも通さず、直接伝える。






1. haldを通して伝える場合。



従来のxorg.confで書いていた設定をhald向けに書き直す必要がある。



設定ファイルは/usr/local/etc/hal/fdi/policy/に置いとけばhaldが見てくれる。



ベースにする設定ファイルは/usr/local/share/hal/fdi/policy/10osvendor/10-x11-input.fdi。



<a href="http://cgit.freedesktop.org/xorg/xserver/tree/config/x11-input.fdi" target="_blank">http://cgit.freedesktop.org/xorg/xserver/tree/config/x11-input.fdi</a>



ここを見ると、xorg.confのオプションをそれぞれ以下の指定に当てはめていけばいいことが分かる。

input.x11_options.XkbRules
input.x11_options.XkbModel
input.x11_options.XkbLayout
input.x11_options.XkbVariant
input.x11_options.XkbOptions

なのでこうなる。

&#60;?xml version=&#34;1.0&#34; encoding=&#34;ISO-8859-1&#34;?>
&#60;deviceinfo version=&#34;0.2&#34;>
&#60;device>
&#60;match key=&#34;info.capabilities&#34; contains=&#34;input.keyboard&#34;>
&#60;!-- If we&#39;re using Linux, we use evdev by default (falling back to
keyboard otherwise). -->
&#60;merge key=&#34;input.x11_driver&#34; type=&#34;string&#34;>kbd&#60;/merge>
&#60;!-- ここから追加 -->
&#60;merge key=&#34;input.x11_options.XkbRules&#34; type=&#34;string&#34;>xorg&#60;/merge>
&#60;merge key=&#34;input.x11_options.XkbLayout&#34; type=&#34;string&#34;>jp&#60;/merge>
&#60;merge key=&#34;input.x11_options.XkbModel&#34; type=&#34;string&#34;>jp106&#60;/merge>
&#60;merge key=&#34;input.x11_options.XkbOptions&#34; type=&#34;string&#34;>ctrl:swapcaps&#60;/merge>
&#60;!-- ここまで追加 -->
&#60;match key=&#34;/org/freedesktop/Hal/devices/computer:system.kernel.name&#34;
string=&#34;Linux&#34;>
&#60;merge key=&#34;input.x11_driver&#34; type=&#34;string&#34;>evdev&#60;/merge>
&#60;/match>
&#60;/match>
&#60;/device>
&#60;/deviceinfo>

名前は適当に/usr/local/etc/hal/fdi/policy/10-x11-input-jp106-capsctrl.fdiとして、haldを再起動してからXを起動すればOK。
